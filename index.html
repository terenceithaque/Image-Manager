Bienvenue sur Image Manager ! Stockez des images provenant de votre disque dur en ligne.

A quoi sert Image Manager ?

Pour pouvoir récupérer vos images, nous aurons besoin de vous identifier. Pour cela, nous vous demandons d'entrer votre nom d'utilisateur, ou de créer un nouvel utilisateur.

Créer un nouvel utilisateur ne supprime pas les autres utilisateurs.
